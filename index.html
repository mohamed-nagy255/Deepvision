Description

Glaucoma is a group of eye diseases that can cause vision loss and blindness by damaging a nerve in the back of the eye called the optic nerve. The symptoms can start so slowly that they may not be noticed.The only way to diagnose glaucoma is a comprehensive dilated eye exam. There is no cure for glaucoma, but early treatment can often stop the damage and protect the vision.


Our main goal:


  1. Detect glaucoma at early stages so patient vision can be protected.
  2. Avoiding the complete blindness caused by glaucoma.

So We created this project to build up an easy , reliable and accurate glaucoma diagnostic system based on deep learning approach as it takes a funds image and then return the result if positive or negative.
